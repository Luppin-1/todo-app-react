# Todo List App 📝

A simple todo list application built with React, Vite, and Tailwind CSS.

## Features
- Add new tasks
- Edit existing tasks
- Delete tasks
- Persistent state management with React Context API
- Clean and modern UI with Tailwind CSS
- Fully responsive design

## Technologies
- React 19
- Vite
- Tailwind CSS
- React Context API
- ESLint

## How to run

```bash
# Clone the repository
git clone https://github.com/your-username/todo-app.git

# Navigate to project folder
cd todo-app

# Install dependencies
npm install

# Start development server
npm run dev
